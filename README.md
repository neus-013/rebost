# Rebost

Una app per a la gestió del rebost, receptes i llista de la compra. Feta amb Flutter.

## Funcionalitats

- Identificació d'usuari amb dades locals per perfil
- Dashboard amb notificacions
- El meu rebost (pròximament)
- Llibres de receptes (pròximament)
- Llista de la compra (pròximament)

## Començar

### Requisits

- [Flutter SDK](https://flutter.dev/docs/get-started/install) >= 3.41
- Dart SDK (inclòs amb Flutter)

### Instal·lació

```bash
flutter pub get
```

### Executar

```bash
flutter run
```
